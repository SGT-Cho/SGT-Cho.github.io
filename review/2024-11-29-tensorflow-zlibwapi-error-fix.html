Could not locate zlibwapi.dll. Please make sure it is in your library path! 오류해결

November 29, 2024
2 min read
Review
TensorFlow 오류해결 zlib CUDA Anaconda Python 팁

tensorflow를 사용하려던 중 계속 오류가 나고 저는 아나콘다 환경을 사용중이라

conda install -c conda-forge zlib

이거를 입력해서 설치를 받아도 계속 같은 오류만 생겼었다,
구글링을 해보니 답이 나와서 공유차 글을 쓴다.

C:\Program Files\NVIDIA Corporation\Nsight Systems 2022.4.2\host-windows-x64\zlib.dll

에 있는 `zlib.dll` 파일을 복사해서

C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.8\bin\zlibwapi.dll

에 붙여넣고 이름을 `zlibwapi.dll`로 바꿔주니 정상 작동했습니다.

(참고: 위 경로는 사용자 환경에 따라 다를 수 있습니다. CUDA 및 NVIDIA Nsight Systems 설치 경로를 확인하세요.)
