Local LLM (LLaMA 3.1 8B + RAG)

로컬 환경에서 LoRA로 파인튜닝한 LLaMA 3.1 8B와 RAG(Retrieval-Augmented Generation)를 결합하여, 개인화된 문서 질의응답 및 요약 시스템을 구현했습니다. Docker와 Kubernetes로 배포 자동화, GPU 자원 최적화, 프롬프트 엔지니어링 경험을 쌓았습니다.

  • Python, PyTorch, LoRA, RAG, Docker, Kubernetes
  • 문서 검색 및 요약, 프라이버시 보장

Building Crack Detection (SegFormer)

  • Python, PyTorch, SegFormer
  • 건설 안전, 이미지 분할

Speech Recognition (Google Cloud STT)

Google Cloud Speech-to-Text API를 활용하여 다양한 음성 데이터를 텍스트로 변환하는 시스템을 개발했습니다. 실시간 스트리밍, 한글 인식, 잡음 환경 테스트를 진행했습니다.

  • Python, Google Cloud STT, Streaming API
  • 음성 인식, 실시간 처리

  • Python, TensorFlow, U-Net
  • 차량 손상 분석, 대회 경험

Real-Time Car Tracking (CCTV)

CCTV 영상을 활용해 실시간으로 차량을 추적하고 속도를 측정하는 시스템을 개발했습니다. 객체 탐지, 트래킹 알고리즘, 속도 계산 로직을 구현했습니다.

  • Python, OpenCV, YOLO, CCTV
  • 실시간 추적, 속도 측정
